# DevOps Portfolio Demo

Bu proje, DevOps becerilerimi göstermek için oluşturulmuş bir portföy projesidir.

## Kullanılan Teknolojiler
- Java 17 + Spring Boot
- Maven
- Docker (containerization)
- GitHub Actions (CI/CD - yakında eklenecek)
- Kubernetes, Terraform, Ansible (sonraki adımlar)

## Local Çalıştırma
```bash
cd devops-demo
mvn spring-boot:run
